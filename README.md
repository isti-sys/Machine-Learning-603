# Machine Learning 603

Repository ini berisi kumpulan tugas mata kuliah **Machine Learning (RE603)** yang dikerjakan selama perkuliahan.

## Struktur Repository

Repository ini dibagi berdasarkan minggu pengerjaan tugas.

## Deskripsi

Setiap folder berisi:

* Notebook atau kode Python
* Dataset yang digunakan (jika ada)
* Penjelasan atau analisis dari tugas yang dikerjakan

Materi yang dipelajari dalam repository ini meliputi:

* Data preprocessing
* Exploratory Data Analysis (EDA)
* Implementasi algoritma Machine Learning
* Evaluasi model
